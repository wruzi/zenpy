Variables & Types

Python is dynamically typed. Use readable names and avoid single-letter variables in production code.

name = "ZenPy"
age = 21
pi = 3.14159
active = True
skills = ["loops", "functions"]
profile = {"level": 4, "xp": 1250}

Control Flow

Use indentation consistently. Prefer clear branches over deeply nested conditions.

if xp >= 1000:
    rank = "Pro"
elif xp >= 300:
    rank = "Intermediate"
else:
    rank = "Beginner"

for index in range(1, 6):
    print(index)

Functions

Use docstrings and type hints for maintainability.

def solve_score(correct: int, speed_bonus: int = 0) -> int:
    """Return XP score for a solved problem."""
    base = 25 * correct
    return base + speed_bonus

print(solve_score(4, speed_bonus=10))

Comprehensions

Comprehensions are concise and fast for mapping/filtering collections.

numbers = [1, 2, 3, 4, 5, 6]
squares = [number ** 2 for number in numbers]
fast = [time for time in [14, 22, 31, 18] if time < 20]

Classes (OOP)

Use classes when behavior and state should live together.

class User:
    def __init__(self, username: str, level: int = 1):
        self.username = username
        self.level = level

    def level_up(self) -> None:
        self.level += 1

u = User("coder")
u.level_up()

Collections

Pick the right type: list (order), set (unique), dict (key-value), tuple (immutable).

tags = {"python", "algorithms", "python"}
# {'python', 'algorithms'}

counts = {"easy": 12, "medium": 7}
counts["hard"] = counts.get("hard", 0) + 1

Exceptions

Catch expected errors and keep exception blocks specific.

try:
    value = int("42")
except ValueError as error:
    print("Invalid number", error)
else:
    print("OK", value)
finally:
    print("Done")

File Handling

Use context managers (`with`) so files are always closed safely.

with open("notes.txt", "w", encoding="utf-8") as file:
    file.write("Practice daily\n")

with open("notes.txt", "r", encoding="utf-8") as file:
    content = file.read()

Modules & Packages

Split code by responsibility and import only what you need.

import math
from datetime import datetime

radius = 5
area = math.pi * radius ** 2
print(datetime.now(), area)

Decorators

Decorators wrap functions for logging, auth checks, caching, and timing.

def logger(fn):
    def wrapped(*args, **kwargs):
        print("Calling", fn.__name__)
        return fn(*args, **kwargs)
    return wrapped

@logger
def greet(name):
    return f"Hi {name}"

Async / Await

Use `asyncio` for I/O-bound tasks (network, file, APIs).

import asyncio

async def fetch_mock():
    await asyncio.sleep(1)
    return "done"

result = asyncio.run(fetch_mock())

Type Hints & Dataclasses

Type hints improve readability and editor support.

from dataclasses import dataclass

@dataclass
class Progress:
    question: int
    xp: int

record = Progress(question=12, xp=430)
